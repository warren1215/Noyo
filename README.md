# Noyo
Coding challenge for Noyo


## Requirements
1) Must have Python 3
2) Run 'pip install -r requirements.txt' from within the directory after downloading the repo.


## Creating the Database
1) Run 'python'
2) 'from Noyo import db'
3) 'db.create_all()'
4) 'quit()'


## Running the server
1) 'python Noyo.py'
